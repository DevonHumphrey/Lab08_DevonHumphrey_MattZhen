## Summary
In this lab, we utilized a multiplexer and demultiplexer to route multiple inputs to one output (multiplexer) and one input to four outputs (demultiplexer). Using Verilog, we implemented a multiplexer and demultiplexer module using ternary operators and concatenation statements. 
## Lab Questions

### In plain English describe the function and use of a multiplexer.
Take in many inputs, route to one output.
### In plain English describe the function and use of a demultiplexer.
Take in one input, route to many outputs.
### What other uses might these circuits have? (Think Shannon’s)
With Shannon’s expansion, you can break a function into parts based on a certain select. This would be similar to splitting the function into two inputs using the select.
